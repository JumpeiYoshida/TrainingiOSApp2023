# TrainingiOSApp2023

## 2023年向けのiOS教育向けアプリのサンプル

## 画面構成

スプラッシュ画面 → お店一覧画面 → お店詳細画面
  
## 画面詳細
### スプラッシュ画面
白いままを予定

### お店一覧画面
リクルートさんのホットペッパーのグルメサーチAPIを使用して東京のお店一覧を表示させる

### お店詳細画面
一覧画面で使用したAPIのレスポンスから「shop」の要素をもらって表示させる
クーポンURLがあれば、そのURLを使ってWebViewを表示できるようにする

## 使用ライブラリ
